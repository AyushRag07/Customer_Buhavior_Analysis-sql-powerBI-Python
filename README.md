# 🛍️ Customer Behavior Analysis

## 📌 Overview
An end-to-end data analytics project focused on understanding customer shopping behavior using transactional retail data.

Using **Python, MySQL, and Power BI**, this project analyzes customer spending patterns, product performance, discount impact, and customer segmentation to generate business insights and support data-driven decision-making.

---

## 🛠️ Tech Stack
- Python
- Pandas
- MySQL
- Power BI
- Jupyter Notebook

---

## ⚙️ Project Workflow
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SQL Business Analysis
- Interactive Dashboard Development
- Business Insights & Recommendations

---

## 📊 Key Insights
- Clothing generated the highest revenue and sales.
- Young Adults contributed the highest overall revenue.
- Subscribers showed stronger purchasing behavior.
- Discounts significantly influenced buying decisions.
- Loyal customers formed a major revenue-driving segment.
